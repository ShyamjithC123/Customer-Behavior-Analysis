# Customer_Behavior_Analysis
Data analytics project showcasing customer shopping behavior analysis using Python, SQL and Power BI

# 📊 Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow using Python, SQL, and Power BI. The goal is to transform raw data into meaningful insights through data cleaning, exploratory data analysis (EDA), SQL-based analysis, interactive dashboards, and professional reporting.

The project covers the entire analytics process, from importing the dataset to presenting the final findings through a Power BI dashboard, a detailed report, and a presentation created using Gamma.

---

## Dataset

The dataset used in this project contains structured data for analysis.

### Dataset Includes

* Multiple records with various attributes
* Numeric and categorical features
* Missing values and duplicate records
* Data suitable for business analysis and visualization

---

## Project Workflow

### 1. Data Loading

* Imported the dataset using Python
* Loaded data into Pandas DataFrame
* Verified data structure and data types

### 2. Exploratory Data Analysis (EDA)

* Examined dataset shape and columns
* Generated descriptive statistics
* Identified missing values
* Detected duplicate records
* Analyzed distributions and relationships
* Created visualizations for better understanding

### 3. Data Cleaning

* Removed duplicate records
* Handled missing values
* Corrected inconsistent data
* Converted data types where required
* Prepared the dataset for analysis

### 4. SQL Analysis

Performed analytical queries using:

* PostgreSQL
* MySQL
* Microsoft SQL Server

Examples of SQL tasks:

* Filtering records
* Sorting and grouping data
* Aggregate functions
* JOIN operations
* Window functions
* Business KPI calculations

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* KPI Cards
* Bar Charts
* Line Charts
* Pie/Donut Charts
* Trend Analysis
* Filters and Slicers
* Interactive Navigation
* Business Performance Insights

### 6. Reporting

Prepared a professional project report including:

* Project objective
* Data cleaning process
* EDA findings
* SQL analysis
* Dashboard screenshots
* Key insights
* Final conclusions

### 7. Presentation

Created a professional presentation using **Gamma** to summarize:

* Project overview
* Methodology
* Analysis process
* Dashboard highlights
* Business insights
* Final conclusions

# How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/ShyamjithC123/your-repository.git
```

### 2. Install Required Python Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### 3. Open the Notebook

Run the Jupyter Notebook or open it in Visual Studio Code.

### 4. Execute SQL Scripts

Run the SQL files using:

* PostgreSQL
* MySQL
* Microsoft SQL Server

### 5. Open Power BI Dashboard

Open the `.pbix` file in Power BI Desktop to explore the interactive dashboard.
